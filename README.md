# PruebaTecnicaCarsales

Prueba técnica: aplicación full-stack que consume la API de Rick and Morty
mediante una arquitectura Backend for Frontend (BFF).

## Estructura

- `/backend` — API en .NET 8 (BFF que consume la API externa).
- `/frontend` — Aplicación en Angular.

> Las instrucciones detalladas de instalación y ejecución están en los
> README de cada carpeta.
